## Algorithm Blocks

### T40.FLG.001 — Filing Requirement Determination
**Input:** `filing_status`, `gross_income`, `age_65_or_older`, `se_net_earnings`, `special_taxes_owed`
**Output:** `must_file`

**2025 Thresholds:**
- Single (under 65): $14,600 | Single (65+): $16,550
- MFJ (both under 65): $29,200 | MFJ (one 65+): $30,750 | MFJ (both 65+): $32,300
- MFS (any age): $5
- HOH (under 65): $21,900 | HOH (65+): $23,850
- QSS (under 65): $29,200 | QSS (65+): $30,750

1. Compare `gross_income` to applicable threshold for `filing_status` and `age_65_or_older`
2. If `se_net_earnings` ≥ $400 → MUST FILE regardless of gross income
3. If `special_taxes_owed` = true (AMT, household employee, etc.) → MUST FILE
4. If received advance premium tax credit → MUST FILE
5. Gross income below threshold AND no special conditions → may not need to file (PASS = no requirement)
6. Gross income above threshold → FAIL ("Taxpayer must file Form 1040")

### T40.STS.002 — Filing Status Determination
**Input:** `marital_status`, `date_of_spouse_death`, `qualifying_dependent`, `paid_household_costs_pct`
**Output:** `correct_filing_status`
1. If `marital_status` = "married as of Dec 31, 2025" → MFJ or MFS
2. If spouse died in 2025 → can still file MFJ for 2025
3. If spouse died in 2023 or 2024 AND `qualifying_dependent` = true AND not remarried → QSS (26% rate structure)
4. HOH: unmarried AND `paid_household_costs_pct` > 50% AND qualifying person lived with taxpayer > 6 months
5. Single: all other unmarried persons
6. Flag if taxpayer is using MFS — note potential disadvantages (no EITC, reduced CTC, etc.)

### T40.INC.003 — Income Completeness and Classification
**Input:** `w2s[]`, `form_1099s[]`, `k1s[]`, `crypto_transactions[]`, `foreign_income`
**Output:** `income_complete`
1. Verify all W-2 Box 1 amounts included
2. For 1099-INT: verify all taxable interest included; tax-exempt interest on Line 2a
3. For 1099-DIV: ordinary dividends on Line 3b; qualified dividends on Line 3a
4. Schedule B required if interest OR ordinary dividends > $1,500 → verify Schedule B attached
5. For K-1s: verify all partnership, S-Corp, trust income included
6. If `crypto_transactions` non-empty → verify each disposition on Form 8949 → if not → FAIL
7. If `foreign_income` > $0 → verify Form 2555 (exclusion) or Form 1116 (credit) attached
8. Social Security benefits: verify Line 6b calculation using combined income formula
9. Any 1099-C received → verify cancellation of debt income included or exclusion documented
10. All income sources accounted for → PASS

### T40.AGI.004 — AGI Adjustments
**Input:** student_loan_interest, hsa_contributions, se_tax_half, se_health_insurance, ira_contributions, educator_expenses
**Output:** `agi_correct`
1. Student loan interest: max $2,500; phaseout $165,000–$195,000 MFJ (2025) → compute correctly
2. HSA: $4,300 self-only / $8,550 family (2025); +$1,000 if 55+; must have HDHP → verify eligibility
3. SE tax deduction: exactly 50% of Schedule SE Line 12 → verify calculation
4. IRA deduction: max $7,000 ($8,000 if 50+); phaseout if covered by workplace plan:
   - Single/HOH: $79,000–$89,000 (2025)
   - MFJ (covered spouse): $126,000–$146,000
   - MFJ (non-covered, spouse covered): $236,000–$246,000
5. Educator expenses: max $300 ($600 if MFJ both educators) → verify cap applied
6. All adjustments within limits and eligibility → PASS

### T40.DED.005 — Standard vs. Itemized Deduction Selection
**Input:** `filing_status`, `itemized_total`, `salt_amount`, `mortgage_interest`, `charitable_contributions`, `medical_expenses`, `agi`
**Output:** `deduction_optimized`
1. Calculate standard deduction for `filing_status` (2025: Single $15,000 / MFJ $30,000 / HOH $22,500)
2. Add additional standard deduction if 65+ or blind
3. Calculate itemized: SALT (capped at $10,000) + mortgage interest + charitable + medical excess
4. Medical: only amount exceeding 7.5% of AGI is deductible
5. If `itemized_total` > standard → recommend itemized; if standard > itemized → recommend standard
6. If taxpayer is MFS → if one spouse itemizes, the other MUST itemize
7. Flag if SALT exceeds $10,000 (cap applies)
8. Deduction choice optimized → PASS; suboptimal choice → FAIL

### T40.CRD.006 — Credit Eligibility and Calculation
**Input:** eitc_eligible, ctc_qualifying_children, aotc_eligible, child_care_expenses, premium_tax_credit_advance, saver_credit_eligible
**Output:** `credits_correct`
1. **EITC:** Must have earned income; no investment income > $11,600 (2025); phaseout by filing status and children count → verify correct credit amount from IRS table
2. **CTC:** $2,000 per qualifying child under 17; $1,700 refundable (ACTC); phaseout: $200K single / $400K MFJ; verify each child's SSN, age, and relationship test
3. **AOTC:** First 4 years of post-secondary education; 40% refundable ($1,000 max); phaseout $80K–$90K single / $160K–$180K MFJ; Form 1098-T required
4. **Child/Dependent Care:** 20-35% of up to $3,000 (1 dep) / $6,000 (2+ deps); must have earned income; care provider tax ID required
5. **Premium Tax Credit:** Reconcile advance payments (Form 1095-A) on Form 8962; repayment cap applies at lower incomes
6. **Saver's Credit:** Contributions to retirement plans; income < $38,250 single / $76,500 MFJ (2025); non-refundable
7. All applicable credits calculated at correct amounts → PASS

### T40.STC.007 — Capital Gains Tax Rate Application
**Input:** `short_term_gains`, `long_term_gains`, `qualified_dividends`, `taxable_income`, `filing_status`, `net_investment_income`
**Output:** `capital_gains_tax_correct`
1. Short-term gains: taxed as ordinary income at marginal rate → verify not using LTCG rate
2. Long-term gains + qualified dividends: apply LTCG rate based on 2025 thresholds:
   - 0%: Single ≤ $48,350 / MFJ ≤ $96,700
   - 15%: Single $48,351–$533,400 / MFJ $96,701–$600,050
   - 20%: Single > $533,400 / MFJ > $600,050
3. Verify Unrecaptured Section 1250 gain taxed at 25% max
4. Collectibles gain taxed at 28% max
5. **NIIT (3.8%):** Apply if MAGI > $200K single / $250K MFJ on lesser of NII or excess MAGI
6. Capital loss limitation: max $3,000 ordinary income offset; verify carryforward on Schedule D
7. Wash sale: flag if substantially identical security purchased within 30 days before/after sale
8. All rates correctly applied → PASS

### T40.AMT.008 — Alternative Minimum Tax Calculation
**Input:** `regular_tax`, `iso_exercise_spread`, `salt_deduction`, `agi`, `filing_status`
**Output:** `amt_correct`
1. Start with regular taxable income; add back AMT preference items:
   - Standard deduction or SALT deduction (add back for AMT)
   - ISO spread at exercise (not sale): add as AMT preference
   - Accelerated depreciation adjustments
2. Subtract AMT exemption (2025: Single $85,700 / MFJ $133,300); phaseout applies above ($609,350 / $1,252,700)
3. Apply AMT rate: 26% on first $239,100 of AMTI; 28% on excess
4. AMT = Tentative Minimum Tax − Regular Tax (if positive)
5. Flag ISO exercise: even if no regular tax, may owe AMT → verify Form 6251
6. AMT credit carryforward from prior year ISO AMT → verify Form 8801
7. AMT correctly computed → PASS

### T40.SEL.009 — Self-Employment Tax and QBI Deduction
**Input:** `se_net_income`, `qbi_amount`, `w2_wages_paid`, `sstb`, `agi`, `filing_status`
**Output:** `se_and_qbi_correct`
1. SE tax: `se_net_income` × 92.35% = SE income; × 15.3% (up to $176,100 SS wage base 2025); × 2.9% above
2. SE tax deduction: 50% of SE tax → verify above-the-line deduction claimed
3. QBI deduction (§ 199A): 20% of lesser of QBI or (taxable income − net capital gains)
4. Below income threshold ($197,300 single / $394,600 MFJ 2025): 20% QBI regardless of entity type
5. Above threshold: apply W-2 wage limitation (greater of 50% of W-2 wages OR 25% of W-2 wages + 2.5% of unadjusted basis of qualified property)
6. SSTB above threshold: QBI deduction phases out completely at $247,300 single / $494,600 MFJ
7. QBI deduction reported on Form 8995 or 8995-A → verify correct form used
8. All calculations correct → PASS

### T40.EST.010 — Estimated Tax / Underpayment Penalty
**Input:** `tax_withheld`, `estimated_payments`, `prior_year_tax`, `current_year_tax`, `agi_prior_year`
**Output:** `underpayment_assessed`
1. Calculate total payments: `tax_withheld` + `estimated_payments`
2. Safe harbor calculation:
   - 90% of 2025 tax liability, OR
   - 100% of 2024 tax (110% if 2024 AGI > $150,000)
3. If total payments ≥ safe harbor amount → PASS (no underpayment penalty)
4. If total payments < safe harbor → FAIL ("Underpayment penalty likely applies; compute Form 2210")
5. Check estimated payment due dates: Apr 15, Jun 16, Sep 15, 2025; Jan 15, 2026
6. Annualized income installment method may reduce penalty for uneven income years

### T40.FRN.011 — Foreign Account and FATCA Reporting
**Input:** `foreign_accounts`, `fbar_required`, `fbar_filed`, `fatca_threshold_met`, `form_8938_filed`
**Output:** `foreign_reporting_compliant`
1. FBAR (FinCEN 114): if aggregate foreign account value exceeded $10,000 at ANY point in 2025 → `fbar_required` = true
   - `fbar_filed` must be true by April 15, 2026 (auto-extension to Oct 15) → if false → FAIL
   - File at FinCEN BSA E-Filing System (separate from tax return)
2. FATCA (Form 8938): if foreign financial assets > $50,000 single / $100,000 MFJ at year-end ($75K/$150K at any point)
   - `form_8938_filed` must be true → if false → FAIL
3. Foreign tax credit (Form 1116) OR exclusion (Form 2555) for income earned abroad → verify claimed
4. All foreign reporting requirements met → PASS

### T40.CRY.012 — Cryptocurrency / Virtual Currency Reporting
**Input:** `crypto_transactions[]`, `mining_staking_income`, `hard_fork_income`
**Output:** `crypto_compliant`
1. For each crypto disposition (sale, exchange, payment for goods/services):
   - Must be reported on Form 8949 with: description, date acquired, date sold, proceeds, cost basis, gain/loss
   - Holding period (short vs. long term) determines tax rate
   - If `each_disposition_reported` = false → FAIL ("Each crypto disposition is a taxable event — Form 8949 required")
2. Mining income: taxable as ordinary income at FMV on date received → verify included in gross income
3. Staking rewards: taxable as ordinary income at FMV on date received → verify included
4. Hard forks creating new tokens: taxable as ordinary income at FMV when received and have dominion/control
5. Like-kind exchange (§ 1031) does NOT apply to crypto after 2017 → flag if claimed
6. Virtual currency question on Form 1040 (first page) must be answered → verify
7. All crypto correctly reported → PASS
