# React & UI Development Interview Questions - TalentVillage.ai

## Technical Interview Questions (Medium to Senior Level)

### 1. React Reconciliation & Virtual DOM
**Q:** Explain how React's reconciliation algorithm works. What is the significance of keys in lists? How does React decide which DOM nodes to update, and how can you optimize re-renders?

### 2. Custom Hooks Design
**Q:** Create a custom `useDebounce` hook and a `useAsync` hook for handling API calls. How would you handle cleanup, error states, and loading states? Discuss dependency arrays and hook rules.

### 3. State Management Architecture
**Q:** Compare React Context, Zustand, and TanStack Query for state management. Design a state architecture for a complex app with auth, user preferences, and server data. When would you use each solution?

### 4. Performance Optimization
**Q:** Your React app is experiencing slow renders and janky scrolling. Walk through your debugging process using React DevTools Profiler. Implement solutions using memoization, code splitting, and virtualization.

### 5. TypeScript Generic Components
**Q:** Create a generic `DataTable<T>` component with TypeScript that accepts any data type, provides sorting, filtering, and pagination. How would you ensure type safety for column definitions and row actions?

### 6. Concurrent Features
**Q:** Explain React 18's concurrent features: automatic batching, transitions, and Suspense. When would you use `startTransition`? How does concurrent rendering improve user experience?

### 7. Form Validation & UX
**Q:** Build a complex multi-step form using React Hook Form and Zod. Implement field-level validation, conditional fields, and proper error display. How would you persist form state across steps?

### 8. Accessibility Implementation
**Q:** Make a custom dropdown menu fully accessible according to WCAG 2.1 AA standards. Implement keyboard navigation, ARIA attributes, focus management, and screen reader support. How would you test it?

### 9. Code Splitting Strategies
**Q:** Design a code splitting strategy for a large React app. How would you implement route-based splitting, component-based splitting, and vendor chunking? Discuss bundle analysis and optimization.

### 10. Server State Management
**Q:** Using TanStack Query, implement optimistic updates for a todo list that syncs with a backend. How would you handle conflicts, cache invalidation, and background refetching?

### 11. Compound Components Pattern
**Q:** Implement a flexible `Tabs` component system using the compound component pattern with Context. Users should compose `<Tabs>`, `<TabList>`, `<Tab>`, and `<TabPanel>` components. How would you ensure type safety?

### 12. Tailwind CSS Architecture
**Q:** Design a Tailwind CSS design system with custom themes, color tokens, and reusable component variants. How would you handle dark mode, responsive design, and avoid className bloat?

### 13. Error Boundaries & Fallbacks
**Q:** Implement a comprehensive error handling strategy using error boundaries. How would you handle async errors, provide user-friendly fallback UIs, and report errors to a monitoring service?

### 14. Animation & Transitions
**Q:** Create smooth page transitions and micro-interactions using Framer Motion or CSS. How would you implement shared element transitions between routes? Discuss performance considerations.

### 15. Testing Strategy
**Q:** Write comprehensive tests for a form component using React Testing Library. How would you test user interactions, async validation, error states, and accessibility? Compare testing approaches.

### 16. WebSocket Real-Time Updates
**Q:** Implement a real-time notification system in React using WebSockets. How would you handle connection status, automatic reconnection, and update the UI without causing re-renders?

### 17. Infinite Scroll & Virtualization
**Q:** Build an infinite scroll component that loads data as the user scrolls. How would you implement virtualization for a list of 10,000+ items? Discuss Intersection Observer and performance.

### 18. Portal & Modal Management
**Q:** Create a modal management system using React Portals. How would you handle focus trapping, ESC key handling, backdrop clicks, and stacking multiple modals? Ensure accessibility compliance.

### 19. React Router Advanced Patterns
**Q:** Implement protected routes, role-based access control, and optimistic UI with React Router v6 loaders and actions. How would you handle authentication redirects and maintain URL state?

### 20. Build Optimization & Vite
**Q:** Your Vite build is slow and the bundle size is too large. How would you analyze the bundle, implement tree shaking, optimize dependencies, and configure chunk splitting? Discuss lighthouse scores.
